# Dokumentasi Projek Discord bot, Kita-Kan-BarBar Discord server. 

## Memulai
ping bot atau lakukan command `b-help` (prefix dapat berganti kapan saja sesuai keinginan owner) 

## Daftar commands
```js

b-help
b-prefix [prefix baru]

b-teswelcome

b-tesbye
b-say [kalimat]
b-sayembed [kalimat]
```
